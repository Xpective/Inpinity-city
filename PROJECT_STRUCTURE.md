# Inpinity City

Inpinity City is the city layer of the Inpinity ecosystem on Base.

It is designed as a shared city with:
- personal plots
- community megaplots
- borderline cooperation zones
- Inpinity / Inphinity faction logic
- layered inactivity and overbuild mechanics
- crowdfunding for community buildings
- future crafting, components, blueprints, and weapons

## Current state
This repository is in pre-implementation setup phase.

## Main folders
- `docs/city/` -> project planning and specs
- `contracts/city/` -> Solidity contracts, scripts, tests
- `site/` -> city frontend
- `subgraph/city/` -> city indexing layer
- `assets/city/` -> wireframes, slotmaps, icons
